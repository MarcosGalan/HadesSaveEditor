## English Version

**HadesEditor** is a premium, open-source save editor for the game **Hades** (Supergiant Games), modernizing and expanding the work of the original project by Zachary Ennenga.

Specifically re-engineered to support **Hades Version 16 (v1.38203+)** by implementing the LZ4 block decompression required for the current save format.

### 📥 DOWNLOADS

Get the latest compiled version for your system in the **Releases** section:

- 👉 [GO TO RELEASES / DOWNLOAD](https://github.com/MarcosGalan/HadesSaveEditor/releases/latest)

### ✨ Features

- **Version 16 Support**: Fully compatible with the latest game updates.
- **Modern UI**: High-definition, dark-themed interface inspired by Hades' aesthetics.
- **Resource Management**: Darkness, Gems, Diamonds, Nectar, Ambrosia, Titan Blood, etc.
- **Weapons & Aspects**: Unlock every weapon and max out all aspects (Zagreus, Achilles, Arthur, etc.).
- **Companions & Relationships**: Edit affinity levels for 14 main NPCs.
- **Keepsakes (Rings/Objects)**: Full star-based level management (Rank 1, 2, 3) and inventory unlock/lock.
- **Game Progress**: Complete quests, unlock mirror talents, and edit run counts.
- **Maximize All**: Instant one-click unlock for everything.

## Versión en Español

**HadesEditor** es un editor de partidas premium y de código abierto para el juego **Hades** (Supergiant Games), modernizando y ampliando el trabajo del proyecto original de Zachary Ennenga.

Re-diseñado específicamente para soportar **Hades Versión 16 (v1.38203+)** mediante la implementación de la descompresión de bloques LZ4 necesaria para el formato de guardado actual.

### ✨ Características

- **Soporte Versión 16**: Totalmente compatible con las últimas actualizaciones del juego.
- **UI Moderna**: Interfaz de alta definición con tema oscuro inspirada en la estética de Hades.
- **Gestión de Recursos**: Oscuridad, Gemas, Diamantes, Néctar, Ambrosía, Sangre de Titán, etc.
- **Armas y Aspectos**: Desbloquea todas las armas y maximiza todos los aspectos (Zagreus, Aquiles, Arturo, etc.).
- **Compañeros y Relaciones**: Edita los niveles de afinidad de 14 NPCs principales.
- **Recuerdos (Anillos/Objetos)**: Gestión completa de niveles por estrellas (Rango 1, 2, 3) y bloqueo/desbloqueo de inventario.
- **Progreso del Juego**: Completa misiones, desbloquea talentos del espejo y edita el número de intentos de escape.
- **Maximizar Todo**: Desbloqueo instantáneo de todo con un solo clic.
